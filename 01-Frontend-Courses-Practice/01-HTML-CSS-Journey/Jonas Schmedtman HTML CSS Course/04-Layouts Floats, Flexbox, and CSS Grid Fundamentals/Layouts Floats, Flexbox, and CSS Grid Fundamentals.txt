============================================
LAYOUTS: FLOATS, FLEXBOX, AND CSS GRID FUNDAMENTALS
============================================

WHAT IS LAYOUT?

Layout is the way text, images, and other content is placed and arranged on 
a webpage.

Layout gives the page a visual structure, into which we place our content.

Building a layout means: arranging page elements into a visual structure, 
instead of simply having them placed one after another (which is the normal 
flow).

Think of it like designing a magazine:
- Without layout: everything is just stacked vertically (like a single column 
  of text).
- With layout: you have headers, sidebars, main content, footers, and images 
  placed in a structured, visually appealing way.

============================================
PAGE LAYOUT VS COMPONENT LAYOUT
============================================

Page Layout:
- The overall structure of the entire webpage.
- Examples: header, main content area, sidebar, footer.
- Controls how the major sections of the page are arranged.

Component Layout:
- The internal structure of individual components/parts of the page.
- Examples: a navigation menu, a product card, a button group, a gallery.
- Controls how elements inside a single component are arranged.

Example:
- Page Layout: A blog page with a header on top, article in the middle, 
  and a sidebar on the right.
- Component Layout: Inside the navigation component, the links are arranged 
  horizontally next to each other.

============================================
THE 3 WAYS OF BUILDING LAYOUTS WITH CSS
============================================

1. FLOAT LAYOUTS (The Old Way)
   - The traditional method for building layouts of all sizes.
   - Uses the CSS float property to position elements.
   - Still used in some older projects, but getting outdated fast.
   - Not ideal for complex layouts.
   - Requires extra work (clearing floats) to prevent layout issues.
   - Example: floating an image to the left so text wraps around it.

2. FLEXBOX (The Modern Way - 1D)
   - A modern, powerful way of laying out elements in a 1-dimensional row 
     or column.
   - Does NOT use floats (no more clearing issues!).
   - Perfect for COMPONENT layouts.
   - Great for: navigation bars, cards, buttons, form elements, galleries.
   - Makes it easy to align, distribute, and reorder items.
   - Works in one direction: either row (horizontal) or column (vertical).

3. CSS GRID (The Powerful Way - 2D)
   - For laying out elements in a fully-fledged 2-dimensional grid.
   - Perfect for PAGE layouts and complex components.
   - Works in two dimensions: rows AND columns simultaneously.
   - Great for: entire page structure, dashboards, complex galleries.
   - Allows you to create grid-based layouts with rows and columns.
